# PlayseEx
ティラノスクリプト/ティラノビルダーのゲームプロジェクト内で使用できる、効果音再生に **ピッチ修正** を追加したプラグイン。

Howler.js ベースの標準実装と互換性を保ちながら、 SE のピッチ修正を実現します。

## 📥 導入方法
- ティラノスクリプト  
公式解説は [こちら](<https://tyrano.jp/usage/advance/plugin>) をクリック。
1. Release から最新バーションの本プラグインをダウンロード。
2. 導入したいティラノスクリプトゲームプロジェクトの `.\data\others\plugin` フォルダに本プラグインを配置。
3. ゲームプロジェクトフォルダの `data\scenario\first.ks` 内に以下の読込処理を記述。
```TyranoScript
[plugin name="playse_ex"]
```

- ティラノビルダー  
[BOOTH](<>) のダウンロードページをご確認ください。

## ✨ 特徴

- 標準の `[playse]` , `効果音再生コンポーネント` と同じ感覚で使用可能
- `pitch` パラメータでピッチ変更が可能

## タグリファレンス

### [playse_ex]
効果音再生（ピッチ修正）

| パラメータ | 必須 | 解説 | 初期値 |
| --- | --- | --- | --- |
| storage | 〇 | 再生する音楽ファイルを指定します。 | |
| buf | | 効果音を再生するスロットを指定できます。すでに指定スロットで再生中の効果音がある場合、その効果音は停止されます。 | `0` |
| loop | | ループするかどうか。`true` または `false` で指定します。 | `false` |
| sprite_time | | 再生する区間を指定できます。開始時刻と終了時刻をハイフン繋ぎで *秒* 単位で指定します。たとえば `6.0-10.1` と指定すると `00:06.00～00:10.100` の 4.1 秒間を再生します。`loop` 属性が `true` の場合、この間をループ再生します。 | |
| clear | | `true` または `false`。`true` を指定すると、他のスロットで再生中の効果音がある場合、そちらもすべて停止します。 | `false` |
| volume | | 再生する音量を指定できます。`0` 〜 `100` の範囲で指定して下さい。 | |
| pitch | | 離調をセント単位で入力。離調は `100` セント = 半音 = 1キー、1200 セント = 1オクターブ。正数でキーを上げ、マイナスでキーを下げる。 | |

## 🧩 使い方

### 半音上げ
```ks
[playse_ex storage="se.mp3" pitch="100"]
```

### スプライト
500ミリ秒の位置から 2200ミリ秒の位置までスプライト

`Web Audio API` の仕様により、標準の `[playse]` タグの `sprite_time` と異なり、単位は **秒** となります。

```ks
[playse_ex storage="se.mp3" sprite_time="0.5-2.2"]
```

## ⚙ 仕様
ティラノ標準の `[playse]` タグの処理が記述された `kag.tag_audio.js` を解析し、**なるべく** 整合性が取れるように拡張したものです。

標準の `[playse]` の音楽再生は `Howler` を採用しておりますが、ピッチ修正非対応のため、こちらでは `Wed Audio API` へ移植しております。

待機、再生スキップのロジック等は標準のまま移植しておりますが、口パク機能関連の処理は外しております。

また、`Web Audio API` の仕様として、ピッチ修正を行うと音の再生速度も変わります。
